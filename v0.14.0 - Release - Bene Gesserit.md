# v0.14.0 Release Notes — "Bene Gesserit: Language Intelligence"

**Release Date:** April 19, 2026  
**Previous Version:** v0.13.3 (Purity Finalized)  
**Status:** ✅ Stable & Production Ready

## 🎯 Executive Summary

v0.14.0 introduces **autonomous language-aware code editing** through LSP (Language Server Protocol) integration. The agent can now validate syntax and imports in real-time before applying changes, dramatically improving code generation quality and reducing validation cycles.

---

## 🚀 Major Features

### ✨ LSP Client Integration (Autonomous Linting)

#### LSPClient — Asynchronous JSON-RPC Transport
- **Language Support:** Ruff (Python) v0.15.10
- **Features:**
  - Real-time syntax validation via `check_file()` method
  - Asynchronous diagnostic capture with background I/O threads
  - Non-blocking file analysis
  - Automatic error recovery and fallback behavior
  - Zero latency impact on agent reasoning loops

#### AgentOrchestrator Integration — Autonomous Validation Loop
- Integrated LSP validation into the core orchestration pipeline
- Real-time syntax & import validation **before** code generation
- Diagnostic feedback integrated into agent reasoning
- Self-correction mechanism for detected linting issues
- Seamless fallback to blind editing if LSP unavailable

**Example Flow:**
```
Agent → Generate Code → LSP Validation → Issues Found?
                             ↓ Yes
                        Generate Fix → Re-validate → Success ✓
```
