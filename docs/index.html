Installation

Python 3.10+ required. Install from PyPI:

# Install globally or in a venv
$ pip install mentask

# Launch in any project directory
$ mentask

On first run, Mentask prompts for your API key (Google, OpenAI, DeepSeek, Groq) and stores it securely in your system keyring.

Technical Core

🔧 Forge Engine (Agentic Tooling)

Dynamic Python tool synthesis. The agent generates code, performs AST-based validation, and hot-reloads modules via importlib. Zero-restart plugin expansion during active sessions.

· AST Validation · Hot-Reloading · Multi-Pass Validation

🔌 MCP Managed Lifecycle

Full implementation of the Model Context Protocol (MCP). Automated server provisioning, JSON-RPC transport, and recursive resource introspection for external data sources and tools.

· JSON-RPC 2.0 · Server Lifecycle · Tool Discovery

👁️ LSP Diagnostic Loop

Bidirectional integration with Ruff LSP. Injects real-time lint feedback and diagnostics into the agent's context for autonomous error correction and adherence to PEP 8.

· Ruff Integration · Real-time Linting · Auto-Fix Cycles

🛡️ Trust Architecture

Hierarchical security model with three-tier directory trust (Permanent, Session, Restricted). Path canonicalization prevents traversal attacks and unauthorized file mutations.

· Path Validation · Canonicalization · OS Keyring Enc

🧠 Context Compression

Advanced token management using sliding window attention and priority-based relevance scoring. Maximizes model context window efficiency for massive mono-repos.

· Token Optimization · Relevance Scoring · Multi-Model Support

📦 Project Isolation (Env)

Isolated .mentask workspaces. Maintains local history, identity overrides, and project-specific tool forge repositories without global pollution.

· Workspace Scoping · Local Artifacts · Tool Isolation

Command Reference

/initBootstraps .mentask/ with SQLite history and project identity
/artifactsIntrospects the forge/ repository for agent-generated tools
/undoPerforms a state rollback of the last filesystem mutation
